Анализ конкурсных списков абитуриентов 

Командный кейс No 3 «Анализ поступления» 

Краткое описание проекта 

Данный проект представляет собой веб-приложение для автоматизации работы приёмной комиссии вуза. Он разработан в рамках выполнения Командного кейса №3 "Анализ поступления" Московской предпрофессиональной олимпиады школьников (профиль "Информационные технологии"). 

Основные функции: 

Загрузка и обновление конкурсных списков абитуриентов из CSV-файлов. 

Автоматический расчёт проходных баллов на 4 образовательные программы с учётом приоритетов абитуриентов и их согласий на зачисление. 

Визуализация данных в виде интерактивных таблиц с возможностью сортировки и фильтрации. 

Формирование итогового отчёта в формате PDF, содержащего проходные баллы, динамику их изменения, списки зачисленных и общую статистику. 

Цель проекта: оптимизировать работу приёмной комиссии, наглядно показать динамику конкурса и предоставить абитуриентам инструмент для анализа своих шансов на поступление. 

  Инструкция по установке и развёртыванию 

Предварительные требования 

Установленный интерпретатор Python. 

Установленная система управления базами данных SQLite. 

Менеджер пакетов pip. 

Шаги для локального запуска 

Клонируйте репозиторий: 

git clone ссылка на репозиторий 


Установите требуемые библиотеки: 

pip install flask 
pip install flask_login 


Запустите сервер разработки: 

python main.py 


Откройте приложение в браузере: Перейдите по адресу: http://127.0.0.1:5000
